Redacta un script de python en un archivo main.py usando el gestor de paquetes uv para crear un archivo de Excel que resuelva el siguiente ejercicio.
Tu objetivo primero es determinar las operaciones necesarias para resolver el problema y justificar cada una de las operaciones realizadas.
Una vez realizado el ejercicio, se deberá crear un script en Python para crear el archivo de Excel con todas las formulas en celdas necesarias para calcular los resultados.
En caso de que no sea posible generar un archivo con formulas, opta por documentar en un archivo README.md todos y cada uno de los pasos para que una persona pueda recrear el resultados siguiendo instrucciones en 
Excel para realizar los calculos. Incluye las formulas de Excel para poder logralo así como las ubicaciones de las celdas donde deberían colocarse.

El trabajo debe ser profesional y estar correctamente documentado.

## INSTRUCCIONES

Para cada ejercicio desarrolla la tabla en Excel con el algoritmo correspondiente e inclúyelas en tus archivos adjuntos.

***

### 1.

Determina el valor de $$x$$ para la siguiente ecuación:

$$
e^x - \cos x = 0
$$

por el **método de bisección**, con una tolerancia:

$$
\text{tol} \le 1 \times 10^{-6}
$$

***

### 2.

Por el **método de Newton-Raphson**, determina la raíz positiva para la ecuación:

$$
x \tan^{-1}\left(\frac{x}{2}\right) + \ln\left(x^2 + 4\right) - 3 = 0
$$

con una tolerancia:

$$
\text{tol} \le 1 \times 10^{-6}
$$

***

### 3.

Determina el valor de $$x$$ por el **método de la secante** para la siguiente ecuación:

$$
\frac{1}{2} + \frac{1}{4}x^2 - x \sen(x) - \frac{1}{2}\cos(2x) = 0
$$

con una tolerancia:

$$
\text{tol} \le 1 \times 10^{-6}
$$

***

### 4.

Determina la solución a la siguiente ecuación por el **método de punto fijo**:

$$
x^3 + 4x^2 - 10 = 0
$$

con una tolerancia:

$$
\text{tol} \le 1 \times 10^{-6}
$$
